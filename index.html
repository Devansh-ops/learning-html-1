10+ HTML tags

These are most, if not all the tags used to make this webpage.

Most of the descriptions are paraprased from W3 School

  1. <!DOCTYPE>
    All HTML documents must start with a <!DOCTYPE> declaration. The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.
  2. <html>
    The <html> tag represents the root of an HTML document. It contains all other HTML elements (except for the <!DOCTYPE> tag)
  3. <head>
    Contains metadata/information for the document
  4. <title>
    Defines a title for the document
  5. <body>
    The <body> tag defines the document's body. It contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  6. <h1> to <h6>
    The <h1> to <h6> tags are used to define HTML headings. <h1> defines the most important heading. <h6> defines the least important heading.
  7. <p>
    Defines a paragraph
  8. <a>
    Defines a hyperlink
  9. <img>
    Defines an image
  10. <br>
    The <br> tag inserts a single line break. It is an empty tag which means that it has no end tag.
  11. <!--...-->
    Defines a comment
  12. <div>
    The <div> tag defines a division or a section in an HTML document.
  13. <form>
    Defines an HTML form for user input
  14. <input>
    Defines an input control
  15. textarea>
    Defines a multiline input control (text area)
  16. <button>
    Defines a clickable button
  17. <select>
    Defines a drop-down list
  18. <optgroup>
    Defines a group of related options in a drop-down list
  19. <option>
    Defines an option in a drop-down list
  20. <fieldset>
    Groups related elements in a form
  21. <legend>
    Defines a caption for a <fieldset> element
  22. <ul>
    Defines an unordered list
  23. <ol>
    Defines an ordered list
  24. <li>
    Defines a list item
  25. <dl>
    Defines a description list
  26. <dt>
    Defines a term/name in a description list
  27. <table>
    Defines a table
  28. <dd>
    Defines a description of a term/name in a description list
  29. <tr>
    Defines a row in a table
  30. <td>
    Defines a cell in a table
